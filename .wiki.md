# Project Summary
LOBOKO is a multi-page HTML/CSS web application designed for community interaction. It allows users to connect, share moments, and access various services, including messaging, notifications, and emergency contacts. The project features a modern, dark-themed interface inspired by the Boo app, enhancing user experience through a mobile-first design.

# Project Module Description
- **Accueil**: Main landing page with welcome messages and community highlights.
- **Découverte**: Explore new content, trends, and personalized recommendations.
- **Messages**: Send and receive messages from friends.
- **Suggestion**: Personalized content suggestions based on user preferences.
- **Entreprise**: Information about companies, including job offers and services.
- **Notifications**: Display user notifications and interactions.
- **Panier**: Manage items for future purchase.
- **Urgences**: Quick access to emergency services like hospitals and police.
- **Menu**: User profile management and app settings.

# Usage
To set up the project:
1. Clone the repository.
2. Navigate to the `app/frontend` directory.
3. Install dependencies using `npm install`.
4. Build the project.
5. Open `index.html` in a web browser to view the application.
