# LOBOKO - Refonte CSS Style Boo

## Design Guidelines

### Design References
- **Boo App (boo.world)**: Thème sombre, cartes arrondies, design épuré, navigation mobile-first
- **Style**: Dark Mode + Minimaliste + App Mobile

### Color Palette
- Background: #0a0a0a (Noir profond)
- Surface: #1a1a1a (Cartes/sections)
- Surface Hover: #252525 (Survol)
- Border: #2a2a2a (Bordures subtiles)
- Accent: #8b5cf6 (Violet - rappel du gradient actuel)
- Accent Hover: #7c3aed (Violet foncé)
- Text Primary: #ffffff (Blanc)
- Text Secondary: #a0a0a0 (Gris clair)
- Text Muted: #666666 (Gris)

### Typography
- Font: Inter, -apple-system, sans-serif
- Headings: font-weight 600-700
- Body: font-weight 400

### Key Component Styles
- Cards: bg #1a1a1a, border 1px #2a2a2a, border-radius 16px
- Buttons: bg #8b5cf6, text white, border-radius 12px
- Inputs: bg #1a1a1a, border 1px #2a2a2a, border-radius 12px
- Navigation: fixed bottom, bg #0a0a0a, border-top #1a1a1a
